# Project Initiation Checklist

Use this checklist during project intake to ensure consistent decisioning and setup.

- [ ] Project summary and objectives documented
- [ ] Assigned project sponsor and project lead
- [ ] Initial scope and success criteria defined
- [ ] Stakeholder list and communication plan created
- [ ] Preliminary risk assessment completed and owner assigned
- [ ] Budget and resource requirements documented or raised
- [ ] Key milestones and timeline (high-level) defined
- [ ] Governance and approval gates identified
- [ ] Repository or docs folder created in docs/projects/

## Acceptance Criteria

- Sponsor and lead confirmed
- Major risks identified with owners
- Stakeholder communication plan assigned

## Owner

Project Lead
