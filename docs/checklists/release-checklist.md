# Release & Deployment Checklist

Use this checklist before each release to ensure required steps are done and risks are mitigated.

- [ ] Release plan documented and approved by Release Manager
- [ ] Release window scheduled with stakeholders
- [ ] Deployment runbook updated and reviewed
- [ ] Rollback plan identified and tested
- [ ] QA and acceptance tests passed
- [ ] Compliance/security checks completed
- [ ] Stakeholder notification list and comms ready
- [ ] Post-release monitoring and owner assigned

## Acceptance Criteria

- All blocking items closed or documented with mitigations
- Rollback plan and monitoring in place

## Owner

Release Manager
